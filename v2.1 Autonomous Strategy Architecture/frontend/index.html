Strategy File Format Reference
Required structure: 
from strategies._template import BaseStrategy, StrategyConfig
from pydantic import Field
import numpy as np, pandas as pd

class MyConfig(StrategyConfig):
    fast_period: int = Field(10, description="Fast EMA Period")
    slow_period: int = Field(50, description="Slow EMA Period")

class MyStrategy(BaseStrategy):
    name         = "My Strategy"     # shown in dropdown
    description  = "What it does"
    config_model = MyConfig

    def on_start(self, data: pd.DataFrame):
        # Pre-compute all indicators ONCE here
        # Store in self._cache dict
        self._cache = { "ema": [...] }

    def on_bar(self, index: int, data: pd.DataFrame):
        # Read from self._cache only — never compute here
        # Return: "BUY", "SELL", "HOLD"
        # Or with SL/TP: ("BUY", sl_price, tp_price)
        if index < 2: return "HOLD"
        return "HOLD"
Signal return values:
  • "BUY" — enter long, no SL/TP
  • "SELL" — enter short, no SL/TP
  • "HOLD" — do nothing
  • ("BUY", 1.0850, 1.0920) — long with SL and TP (absolute prices)
  • ("SELL", 1.0920, 1.0850) — short with SL and TP
  • ("BUY", None, 1.0920) — long with TP only
Important rules:
  • All heavy computation goes in on_start()
  • on_bar() must only READ from self._cache
  • SL/TP values are absolute price values, not pip distances
  • The data in on_bar() contains only bars up to current — no look-ahead
  • Allowed imports: numpy, pandas, math, typing, pydantic, strategies._template
